# TableTest Reporter

TableTest Reporter is a tool for generating documentation from [TableTest](https://github.com/nchaugen/tabletest) results. It can produce documents in AsciiDoc and Markdown format ready to be published by a static site generator or documentation publishing tool.


## Features

- Generates documentation of TableTest methods and test classes
- Generates table of contents for easy navigation of documentation
- Built-in support for AsciiDoc and Markdown output formats
- Allows customization of AsciiDoc and Markdown output
- Supports custom mapping to other output formats
- Plugins for Maven and Gradle to generate documentation as part of a build
- Integrates with static site generators and documentation publishing tools


## Templates

TableTest Reporter uses [Pepple Templates](https://pebbletemplates.io) as a templating engine. It allows both easy adjustment to provided templates for AsciiDoc and Markdown output, and the ability for the user to create custom templates for other output formats.
